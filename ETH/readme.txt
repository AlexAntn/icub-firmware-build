
The folder organization is as follows:

--ETH
    |--EMS                  contains whatever is required for the programming of the EMS board. refer to the nested readme.txt for more details.
    |    
    |--MC4PLUS              contains whatever is required for the programming of the MC4PLUS board. refer to the nested readme.txt for more details.
    |
    |--MC2PLUS              contains whatever is required for the programming of the MC2PLUS board. refer to the nested readme.txt for more details.
 
 
In file howto-eth-boards-fundamentals.txt there are some fundamental information about ETH boards.

In file howto-eth-boards-programming.txt there instructions about how to program the ETH boards and how to perform FW update of ETH boards and of 
the CAN boards attached to them.
